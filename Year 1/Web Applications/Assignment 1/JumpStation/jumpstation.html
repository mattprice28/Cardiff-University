About JumpStation

What is JumpStation?

JumpStation was the world's first web crawler. Very simply a web crawler takes key words and 'crawls' through existing web pages looking for matches, returning a series of hyperlinks as results.

While today this concept is fundamental to even the most basic of web users, before the introduction of JumpStation the web was a very difficult place to navigate.

Life Before JumpStation

So how did early web users find their way around? One way to navigate to different web pages was to type the URL, sound simple? Well think again, this was before your web browser could correct errors and give suggestions, a single spelling mistake or typo would lead to ... well nothing.

An Early Browser

The first site recognisable as a modern day browser was Mosaic. 'Mosaic is an Internet-based global hypermedia browser that allows you to discover, retrieve, and display documents and data from all over the Internet.' (NCSA 1993) As pictured above Mosaic contained a 'What's New' page, this was a chronological list of hyperlinks to available websites

The Problem with Mosaic

While Mosiac offered a more elegant solution than the painstaking manual entry of URL's it too had it's limitations. The biggest flaw of Mosaic was that it had to be maintained manually, Anyone creating a new website would have to write to the team in charge at the University of Illinois for their URL to be added to the what's new page. If mistakes were made, or if websites were updated, this URL is no longer valid and becomes a dead link
